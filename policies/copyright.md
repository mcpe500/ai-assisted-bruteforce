# Copyright Policy

## Repository content

All original content in this repository (code, documentation, prompts, templates) is created by the project contributors.

## Third-party content

### Papers and articles

* Do not commit full copyrighted PDFs to this repository.
* Store metadata, abstracts, and your own summaries only.
* Use DOI links and URLs to reference original sources.
* If a paper is open-access (CC-BY, CC0, or similar), you may store a link but still prefer linking over copying.

### Code and libraries

* Any third-party code must have a compatible open-source license.
* Document the license for each dependency.
* Do not include code with licenses that restrict the intended use of this project.

### Data and datasets

* Follow the dataset's license terms.
* Document the source and license for any dataset used.
* Do not redistribute data in violation of its license.

## Agent-generated content

Content generated by AI agents (summaries, notes, code, simulations) during the course of this project is part of the project output.

Agents should:
* Summarize in their own words, not copy verbatim from sources.
* Attribute ideas and findings to their original sources.
* Clearly distinguish between agent-generated content and sourced content.

## Fair use considerations

Short quotations for the purpose of criticism, commentary, or research may be acceptable under fair use doctrines, but:
* Keep quotations brief and attributed.
* Do not reproduce substantial portions of copyrighted works.
* When in doubt, summarize instead of quoting.
