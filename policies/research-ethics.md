# Research Ethics Policy

## Principles

1. **Respect intellectual property.** Papers, code, and data are protected by copyright and licenses.
2. **Use open-access sources.** Prefer repositories and journals that explicitly provide open-access content.
3. **Do not bypass paywalls.** If content is behind a paywall and you don't have legitimate access, use metadata and abstracts only.
4. **Cite everything.** Every paper, equation, algorithm, and concept note must include proper attribution (authors, title, DOI, URL).
5. **Summarize in your own words.** Do not copy-paste large sections of papers. Summarize and paraphrase.
6. **Distinguish fact from speculation.** Clearly label what is proven, what is assumed, and what is speculative.
7. **Respect rate limits.** Do not mass-scrape websites or APIs. Use official APIs where available.
8. **Track provenance.** Every piece of knowledge should be traceable to its source.

## Allowed sources

* arXiv and other preprint servers
* Open-access journals (Optica, Nature Communications, PLOS, etc.)
* Publisher-provided open-access PDFs
* DOI metadata from CrossRef, Semantic Scholar, etc.
* Abstracts from any source
* User-provided PDFs (assumed to be legally obtained)
* Government and institutional reports (public domain)
* Wikipedia and other open-knowledge references (cite as starting point, not primary source)

## Disallowed sources

* Sci-Hub or similar unauthorized PDF repositories
* Mass-downloaded torrents of copyrighted papers
* Paywall-circumventing browser extensions or scripts
* Scraped content from sites that prohibit scraping in their ToS

## When a PDF is unavailable

1. Store the metadata (title, authors, year, venue, DOI, URL).
2. Store the abstract if publicly available.
3. Write a summary based on the abstract and any publicly available information.
4. Mark the access type as `abstract_only`.
5. Flag it for manual review — the user may have access.

## Data handling

* Do not store full copyrighted PDFs in this repository.
* Store metadata, summaries, and extracted (non-substantial) quotes with attribution.
* If the user provides a PDF, store the notes but not the PDF itself (or store it only if the user explicitly instructs).
